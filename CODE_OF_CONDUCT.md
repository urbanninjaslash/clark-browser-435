# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short:

- Be respectful and inclusive
- Disagree on technical merits, not personal attacks
- Focus discussion on what's best for the project and its users
- No harassment, discrimination, or abuse — period

Violations: report to **conduct@clarklabs.ai**. Maintainers will respond
within 7 days. Consequences range from a private warning to a permanent ban
from the project depending on severity.

— Clark Labs Inc.
